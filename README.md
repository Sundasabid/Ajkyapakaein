# Aaj Kya Pakayen?

A smart Flutter-based meal suggestion application designed to simplify daily cooking decisions through intelligent recommendations and personalized filtering.

The app uses a database of 200+ dishes and suggests the most suitable meal based on user preferences, weather conditions, budget, freezer availability, and cooking history.

---

## Features

* Smart meal recommendation system
* 200+ dishes managed through Firebase
* Weather and budget-based filtering
* Veg, non-veg, and rice category selection
* Freezer availability support
* Prevents repetition of recently cooked meals
* Alternative dish suggestions
* Cooking history tracking
* Secure user authentication
* Smooth and modern UI experience

---

## How It Works

Users answer a few quick questions such as:

* Current weather
* Available budget
* Freezer items
* Preferred meal category

The app then intelligently filters dishes and recommends the best possible option while avoiding meals cooked within the last 7–10 days.
